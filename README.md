# Magui FITT PRO

Aplicación web simple y responsive para seguimiento físico personal.

## Tecnología

- Google Apps Script
- Google Sheets
- Google Drive
- HTML5
- CSS3
- JavaScript ES6
- Google Charts

## Archivos

- `appsscript.json`
- `Code.gs`
- `index.html`
- `style.html`
- `script.html`

## Instalación manual

1. Crear un Google Sheets nuevo.
2. Abrir `Extensiones > Apps Script`.
3. Crear/pegar los archivos del proyecto.
4. Ejecutar `inicializarProyecto`.
5. Aceptar permisos.
6. Publicar como Web App.

## Hojas creadas automáticamente

- Perfil
- Mediciones
- Fotos
- Configuracion
- Frases
